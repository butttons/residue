# @residue/docs

Public documentation site for residue, hosted at [residue.dev](https://residue.dev).

Built with Astro, styled with Tailwind CSS, deployed to Cloudflare Pages.

## Development

```bash
pnpm run dev       # local dev server
pnpm run build     # production build
pnpm run deploy    # build + deploy to Cloudflare Pages
```
